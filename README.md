# 预加载服务接口 demo

## demo运行方式
```bash
$ yarn
$ yarn start
```
或
```bash
$ npm i
$ npm start
```
访问地址 http://localhost:5555/

访问请求(GET)
```bash
http://localhost:3000/start_package?time=XXX&&sign=XXX&&type=XXX&&id=XXX&&email=XXX
```
